# Example app with Jest tests inside a NextJS TypeScript app

## How to use

Use nvm or a node version > 10.0

Setup the api key on a `now-secrets.json` file:

```json
{
  "@openex-api-key": OPENEX_API_KEY
}
```

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

## Run Jest tests

```bash
npm run test
# or
yarn test
```
